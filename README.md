<h1>Dashboard de Vendas em Power BI 💹</h1>

Este repositório contém um dashboard em Power BI que exibe dados de vendas, lucro da empresa, vendas por país e outros indicadores chave. O objetivo deste projeto é fornecer uma visão clara e concisa do desempenho de vendas de uma empresa, utilizando os dados fornecidos para insights que coloaborarão para resultados positivos no saldo da empresa.

<h2>Visão Geral 📊</h2>
O dashboard apresenta as seguintes visualizações e métricas:


<b>Total de Vendas:</b> Exibe o valor total das vendas realizadas em um determinado período.

<b>Lucro da Empresa:</b> Mostra o lucro bruto e líquido da empresa.

<b>Vendas por Países:</b> Um gráfico detalhado das vendas distribuídas por diferentes países.

<b>Desempenho Mensal:</b> Análise do desempenho de vendas e lucro mês a mês.

<b>Top Produtos:</b> Lista dos produtos mais vendidos.


<h2>Pré Requisitos 💻</h2>
Para visualizar e interagir com o dashboard, você precisará do seguinte software:


[Power BI Desktop](https://www.microsoft.com/pt-br/download/details.aspx?id=58494)

<h2>Como Usar ❓</h2>
Clone este repositório em sua máquina local:


git bash

Copiar código
git clone [https://github.com/seu-usuario/First-Project-PowerBI.git](https://github.com/Lucasrof/First-Project-PowerBI.git)

Abra o Power BI Desktop.

No Power BI Desktop, clique em "Arquivo" -> "Abrir" e selecione o arquivo <b>Relatório de Vendas - Primeira Análise.pbix</b> do repositório clonado.
